// Конфигурация wifi с помощью wpa_supplicant

1. Проверим не заблокировано ли питание беспроводной карты ядром:
# rfkill list

2. В случае если заблокирована, то введем след. команду:
# rfkill unblock wifi или all

3. Посмотрим запущен установлено ли "up" на нашем интерфейсе:
# ip link show wlan0

4. Если нету, тогда пропишем команду:
# ip link set wlan0 up

5. Скопируем конфиг в нужное место: 
# cp /usr/share/doc/wpa_supplicant/wpa_supplicant.conf /etc/wpa_supplicant/wpa_supplicant.conf

6. Запускаем wpa_cli и прописываем следующее:
> scan
> scan_results
> add_network
> set_network 0 ssid "MYSSID"
> set_network 0 psk "passphrase" 
	* если пароль не стоит: 
	> set_network 0 key_mgmt NONE
> enable_network 0
> save_config
> quit
